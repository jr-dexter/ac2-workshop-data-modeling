## Requirements

In order to start the application provided for this lab, it's recommended to have:

- Python 3.11 or 3.12
- invoke

## Lab Commands

To simplify the process, there is a CLI provided to be able to easily manage the lab using Invoke. List the available Invoke tasks.

```shell
invoke -l
```
```shell
Available tasks:

  lab1-start     Start Lab1.
  lab1-destroy   Destroy Lab1.
  lab1-load      Load devices into Lab1.
```

:::info
Omitted non-lab1 related Invoke tasks.
:::
