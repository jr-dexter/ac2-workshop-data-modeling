## Requirements

In order to start the application provided for this lab, it's recommended to have :
- Python 3.11 or 3.12
- Docker or Docker Desktop
- invoke

Additionally you will have to install [DB Browser for SQLite](https://sqlitebrowser.org/dl/), or you can install the [SQLite viewer plugin for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=qwtel.sqlite-viewer&ssr=false#overview)

## Lab Commands

to simplify the process, there is a CLI provided to be able to easily manage the lab using Invoke. List the available Invoke tasks.

```shell
invoke -l
```
```shell
Available tasks:

  lab2-destroy   Destroy Lab2.
  lab2-load      Load devices into Lab2.
  lab2-start     Start Lab2.
```

:::info
Omitted non-lab2 related Invoke tasks.
:::
